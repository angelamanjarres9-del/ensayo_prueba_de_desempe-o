# Fusión Gourmet — Gastronomic Festival Landing Page

## Description
"Fusión Gourmet" is a gastronomic festival that celebrates creative cooking and unique culinary experiences.
This landing page was built to show the festival schedule, the featured chefs, a photo gallery and contact information.
The design uses dark tones, gold colors and elegant typography to match the high-end feel of the event.
The main goal is to attract food lovers and get them to buy their tickets.

## Technologies used
- HTML5 — page structure using semantic tags like header, main, section, article, nav and footer
- CSS3 — styles, colors, fonts, Flexbox, Grid layout and media queries for mobile
- Font Awesome — icons for social media links in the footer

## Project structure
- restaurante.html — main HTML file with all the page content
- styleFES.css — all styles: colors, layout, typography and responsive design
- readme.md — project documentation

## Sections included
- **Header** — festival logo and navigation menu
- **Hero** — main impact section with a call to action button
- **Schedule** — tasting and workshop schedule in a table
- **About** — festival concept and philosophy
- **Gallery** — food photo gallery
- **Footer** — contact info and social media links

## How to view locally
Option 1 — open directly:
Just open `restaurante.html` in your browser.

Option 2 — local server:
1. Open a terminal in the project folder
2. Run: `python3 -m http.server 5500`
3. Go to: `http://localhost:5500/restaurante.html`
